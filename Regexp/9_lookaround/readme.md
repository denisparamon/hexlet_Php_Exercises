## solution
Напишите регулярное выражение, которое находит подстроку номер 1, за которой следует подстрока номер 2:

Описание подстрок:

80
: и хотя бы один или более символов, не входящих в класс a-z
Используйте позитивный просмотр вперед.

// match
['80:8080, 80:!@#$'];

// not match
['80:', '80', '80:d123f'];
